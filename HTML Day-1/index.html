Q4. Difference between block and inline ?

Block level element always start from new line and takes the whole width of the screen but in case of line level element it do not occupy unnecessery space and it always start from where previous line end. In css you can change the height and width of block level element but you can not change the height and width of inline level element.

Attributes gives the additional information about the tag, it should be written in lower case letters only .It should be given in the opening tag always. Multiple attributes can be used in a particular tag.
